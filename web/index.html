Agent Foundry

A curated runtime orchestrator for AI coding assistants. Plan → execute → verify. Skills, agents, and a local daemon — all MIT, all local.

# Install
$ curl -fsSL https://raw.githubusercontent.com/youcisla/Agent-Foundry/main/install.sh | bash
# Use in Claude Code
$ /af "build a react component"
$ /plan "audit this API"
# Or use the CLI
$ agent-foundry plan "kill generic AI slop"

What it is

🧠 Skills

Disciplines the model applies to its own work. How to think, not what to know. Each skill ≤150 lines, ≤8 KB.

🤖 Agents

Roles the orchestrator can dispatch — critic, planner, verifier. Each with a model tier and tool allowlist.

⚙️ Orchestrator

Local daemon that ranks, dispatches, executes, logs, and judges every run. FastAPI, SQLite, lazy-started.

Install in 30 seconds

curl | bash git clone profiles
# One-line install (macOS / Linux)
curl -fsSL https://raw.githubusercontent.com/youcisla/Agent-Foundry/main/install.sh | bash

# Set your API key
export ANTHROPIC_API_KEY=sk-...
# Clone and install in editable mode
git clone https://github.com/youcisla/Agent-Foundry.git ~/.agent-foundry
cd ~/.agent-foundry
pip install -e .
agent-foundry init

# Symlink Claude Code plugin
ln -s "$PWD" ~/.claude/plugins/agent-foundry
# Minimal — skills only, no daemon
AF_PROFILE=minimal curl -fsSL ... | bash

# Core — skills + daemon (default)
AF_PROFILE=core curl -fsSL ... | bash

# Full — skills + daemon + hooks
AF_PROFILE=full curl -fsSL ... | bash

Requires Python 3.10+ and an Anthropic or OpenAI API key.

Commands

CommandWhat it does
agent-foundry plan "..."Rank skills for a prompt
agent-foundry run "..."Plan + execute the top-ranked skill
agent-foundry execute <id> "..."Run a specific skill
agent-foundry doctorHealth-check config, index, daemon, API key
agent-foundry statusRouting accuracy, fallback rate, average cost
agent-foundry consult "..."Recommend skills for a need
agent-foundry cost-reportToken and time estimates per skill
agent-foundry indexRebuild the skill index
agent-foundry serveStart the daemon (lazy-started on first command)

Architecture

/af <prompt> /plan <prompt> │ │ ▼ ▼ ┌──────────────────────────────────┐ │ Agent Foundry daemon │ │ FastAPI · planner · judge │ │ executor · budget guard │ │ SQLite log · indexer │ └──────────────────┬───────────────┘ │ ▼ ┌──────────────────────────────────┐ │ Skill catalog + Agent catalog │ │ references/ · scripts/ · │ │ hooks/ · executions.db │ └──────────────────────────────────┘

The daemon is lazy-started by the CLI. Everything runs locally. Your data stays in ~/.config/agent-foundry/executions.db.
