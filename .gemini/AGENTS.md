# Agent Foundry — Gemini CLI Guide

Gemini CLI has no formal plugin system. After install:
1. Symlink skills: `bash scripts/install.sh --harness=gemini-cli`
2. Use `/af` or `/plan` commands as documented in AGENTS.md
3. The daemon (`agent-foundry serve`) works independently
